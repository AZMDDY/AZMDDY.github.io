## 思路

根据题目，递归是最直接的方法，但我们不难发现递归中包含了重复计算，且当前值和之前计算的值有关，可以使用动态规划来实现。

## 实现

### C++

```cpp
class Solution {
public:
    int fib(int N)
    {
        if (N <= 1) {
            return N;
        }
        vector<int> dp(N + 1, 0);
        dp[0] = 0;
        dp[1] = 1;
        for (int i = 2; i <= N; i++) {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        return dp[N];
    }
};
```

### Python

```python
class Solution:
    def fib(self, N: int) -> int:
        if N <= 1:
            return N
        dp = [None] * (N + 1)
        dp[0] = 0
        dp[1] = 1
        for i in range(2, N + 1):
            dp[i] = dp[i - 1] + dp[i - 2]
        return dp[N]
```
